## Decisions
- **Default accent = Apple blue** (`Color(red: 0.0, green: 0.48, blue: 1.0)`) not Penumbra's brand orange. Rationale: NetworkQuality's entire legacy UI used `Color.accentColor` = system blue; switching to orange would make Phase 2's `.accentColor(ā€¦)` wiring a visual shock. Kept as single-line edit in `Theme/Theme.swift:11` for future revision.
- **CropBatch FCPToolbarButtonStyle variant** (non-binding `var isOn: Bool`) over Penumbra's `@Binding var isOn`. Simpler to reason about, works fine with `PaneToggleButton` passing `isOn` as value (SwiftUI still re-renders on binding changes at the wrapping view).
- **Keep `.pickerStyle(.segmented)` in SettingsView** for now ā€” not reachable from main navigation (case 3 has no sidebar row in current layout), so low user impact. Will get proper generic helper before ship.
- **Version bumped to 1.0.3 mid-migration** rather than after Phases 4ā€“7 complete. Rationale: user asked for DMG + notarization now; Phases 1ā€“3 + picker sweep are shippable visual improvements on their own. Phases 5ā€“6 (dark-mode audit of 477 color refs + full verification) can land as 1.0.4 or folded into 1.0.3 before ship.
